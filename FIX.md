# Исправление ошибки 500

## Проблема
Ошибка 500 при загрузке страницы из-за проблем с доступом к файлам node_modules в sandbox окружении.

## Решение

Выполните следующие команды в вашем терминале:

```bash
cd /Users/aleksejkrenicyn/Dashboard

# 1. Остановите сервер (Ctrl+C)

# 2. Очистите все кеши
rm -rf .next node_modules package-lock.json

# 3. Переустановите зависимости
npm install

# 4. Запустите сервер
npm run dev
```

## Альтернативное решение (если проблема сохраняется)

Если ошибка 500 все еще появляется, попробуйте:

```bash
# Используйте yarn вместо npm
npm install -g yarn
yarn install
yarn dev
```

## Проверка

После запуска сервера:
1. Откройте http://localhost:3000
2. Должен быть редирект на /login
3. Если видите белый экран, откройте консоль браузера (F12) и проверьте ошибки

## Если проблема в правах доступа

```bash
# Исправьте права на node_modules
sudo chown -R $(whoami) node_modules
sudo chmod -R u+r node_modules
```
