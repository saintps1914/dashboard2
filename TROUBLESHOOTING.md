# Устранение проблем с белым экраном

## Шаги для диагностики:

1. **Откройте консоль браузера** (F12 или Cmd+Option+I)
   - Проверьте вкладку "Console" на наличие ошибок
   - Проверьте вкладку "Network" - все ли файлы загружаются

2. **Проверьте, что сервер запущен:**
   ```bash
   npm run dev
   ```
   Должно быть сообщение: "Ready on http://localhost:3000"

3. **Проверьте, что TailwindCSS работает:**
   - Откройте http://localhost:3000/login
   - Если видите белый экран, откройте DevTools → Elements
   - Проверьте, применяются ли классы TailwindCSS

4. **Очистите кеш и перезапустите:**
   ```bash
   rm -rf .next
   npm run dev
   ```

5. **Проверьте версию Node.js:**
   ```bash
   node --version
   ```
   Должна быть версия 18 или выше

## Если проблема сохраняется:

1. Проверьте логи сервера в терминале
2. Убедитесь, что все зависимости установлены: `npm install`
3. Попробуйте открыть напрямую: http://localhost:3000/login
